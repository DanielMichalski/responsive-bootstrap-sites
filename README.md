Responsive Bootstrap Sites
==========================

This project is Spring MVC + Bootstrap + Tiles application

It contains some useful configuration items:

- Spring MVC
- Apache Tiles 3.0.5
- Bootstrap 3.2.0
- JQuery 1.11.0
- Log4j
